Opening the webpage

The webpage must be opened on a web server to operate properly. Click the webserver batch file to launch the local webserver. If this does not work, then you will need to download Python
You can check if you have Python installed by opening Command Prompt and typing "python" or "py". 
If you see the version number then you have it installed, otherwise you will need to download it. 
If you do not have Python installed on your computer, the Windows store should automatically open to allow you to download Python. You can also download it from https://www.python.org/downloads/
Once Python is installed, in Command Prompt, change the directory to the folder the webpage is located in by typing "cd [file path index.html is located]" 
Next start the webserver by typing: "python -m http.server" or "py -m http.server" depending on which version of Python you have. 
Navigate to the folder path in your preferred internet browser to open the webpage. 


Updating the data

Update the data in the excel files located in the data folder.
Save them as a webpage (.htm) with the same name. Make sure the first letter is capitalized.
The data should automatically updated in the webpage.
