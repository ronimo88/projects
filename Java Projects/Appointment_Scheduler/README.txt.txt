Title: Appointment Scheduler

Purpose: The purpose of this application is to add and remove customers that utilize the business's services and to schedule appointments for the customers, contacts, and users.

Author: Ronald Morrison
Email: rmor236@wgu.edu
Student Application Version: 1.1
Date: 25 Jan 2022

IDE: IntelliJ IDEA 2021.1.1 (Community Edition)
JDK: Java SE 11.0.11
JavaFx: JavaFX-SDK-11.0.2

Run Instructions: 
   Open the project in IntelliJ and click 'Run' in the toolbar and click 'Run 'Main''
   Login with User Name: test, Password: password

   Click the Appointments tab to view a table of all appointments.
   Click the Month button to view all appointments in the current month.
   Click the Week button to view all appointments in the current week.	
   Click the All button to view all appointments.
   Choose a contact from the filter by contact to show a schedule of all appointments for that contact in the table.
   Click Add to schedule a new appointment. Fill in all details and click submit to schedule it.
   Select an appointment and click Edit to change appointment details. Edit the details and click submit to confirm.
   Select an appointment and click Delete to delete an appointment.
   Click Appointment Report to go to the Appointment Report screen
   Select a type and month and click submit to view the number of appointments of the selected type in the selected month.
   Click the back button to go back to the home screen.

   On the home screen, click the Customers tab to view a table of all customers.
   Click Add to add a new customer. Fill in all details and click submit to add the customer.
   Select a customer and click Edit to change customer details. Edit the details and click submit to confirm.
   Select a customer and click Delete to delete the customer.
   Click the Customer Location Report button to go the the Customer Location Report screen.
   Select a country and division and click submit to view the number of customers in the selected location.
   Click the back button to go back to the main menu.

   Click Logout to logout.
   Click Exit to quit the application.
   
Additonal Report: in the Customers tab, click the 'Customer Location Report' button. Select a country and division to view the number of customers in that location. 

MySQL Connector: mysql-connector-java-8.0.25
